# Цель работы

## Основная цель

Изучение возможностей LaTeX по созданию и форматированию таблиц.

# Ход выполнения

## Компиляция исходного файла

Я открыл файл `tables.tex` в текстовом редакторе и выполнил его компиляцию с помощью команды `pdflatex`.

В процессе компиляции использовались:
- TeX Live 2026
- класс документа `article`
- пакеты для работы с таблицами: `array`, `booktabs`, `siunitx`, `tabularx`, `multirow`, `threeparttable`, `longtable`

![Компиляция tables.tex](image/01.png)

## Титульный лист и простые таблицы

На первой странице сгенерированного PDF я получил титульный лист и простые таблицы с использованием окружения `tabular`.

Я создал два вида простых таблиц:
- таблица с вертикальными и горизонтальными линиями
- таблица без вертикальных линий

![Титульный лист и простые таблицы](image/02.png)

## Таблицы с пакетом booktabs

Я протестировал пакет `booktabs`, который создаёт профессионально выглядящие таблицы с помощью команд `toprule`, `midrule` и `bottomrule`.

Затем я попробовал разные виды линий:
- разная толщина линий с помощью необязательных аргументов
- частичные линии с помощью `cmidrule`
- добавление пространства между строками с помощью `addlinespace`

![Таблицы с booktabs](image/03.png)

## Объединение ячеек

С помощью команды `multicolumn` я реализовал горизонтальное объединение нескольких столбцов. Это позволило мне создать общий заголовок для всей таблицы и объединить две ячейки в одной из строк.

С помощью пакета `multirow` я выполнил вертикальное объединение ячеек.

![Объединение ячеек](image/04.png)

## Комбинация объединений и фиксированная ширина

Я также попробовал комбинировать горизонтальное и вертикальное объединение в одной таблице.

Затем я изучил столбцы с фиксированной шириной типа `p`. Этот тип позволяет тексту автоматически переноситься внутри ячейки, что удобно для длинных описаний.

![Комбинация объединений и тип p](image/05.png)

## Вертикальное выравнивание и пользовательские типы

Я протестировал типы `m` и `b` для вертикального выравнивания:
- тип `m` выравнивает текст по центру ячейки
- тип `b` выравнивает текст по низу ячейки

Также я создал свои типы столбцов с изменением шрифта: жирный, курсив и мелкий шрифт.

![Типы m, b и пользовательские типы](image/06.png)

## Числовое выравнивание с siunitx

С помощью пакета `siunitx` я научился выравнивать числа по десятичной точке. Тип столбца `S` автоматически выравнивает все числа так, чтобы десятичные запятые находились друг под другом.

Это особенно полезно для таблиц с финансовыми или статистическими данными.

![Числовое выравнивание](image/07.png)

## Таблицы с фиксированной шириной (tabularx)

Я использовал пакет `tabularx` для создания таблиц с заданной общей шириной. Тип столбца `X` автоматически растягивается, чтобы заполнить оставшееся пространство.

Я попробовал разные варианты: таблицу с тремя колонками и таблицу с четырьмя колонками разной ширины.

![Таблицы tabularx](image/08.png)

## Многостраничные таблицы (longtable)

Для длинных таблиц, которые не помещаются на одну страницу, я применил пакет `longtable`. Этот пакет автоматически разбивает таблицу на несколько страниц и повторяет заголовки на каждой новой странице.

Я создал таблицу с двадцатью строками, которая занимает две страницы.

![Многостраничные таблицы longtable](image/09.png)

## Таблицы с примечаниями и цветные таблицы

С помощью пакета `threeparttable` я добавил сноски к таблице. Это удобно для указания источников данных или дополнительных пояснений.

С пакетом `xcolor` я создал цветную таблицу с чередованием цветов строк, что улучшает читаемость.

![Таблицы с примечаниями, цветные таблицы и вывод](image/10.png)
